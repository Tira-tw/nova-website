# Nova Launcher

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)]()
[![Language](https://img.shields.io/badge/language-HTML%2FCSS%2FJS-orange.svg)]()
[![License](https://img.shields.io/badge/license-MIT-green.svg)]()

Nova Launcher is a modern launcher built specifically for racing game enthusiasts. We are dedicated to providing a stable, seamless, and boundary-free racing experience.

---

## 🚀 Key Features
* **Modern Interface**: Crafted with Tailwind CSS to deliver an immersive Dark Mode experience.
* **Lightweight Architecture**: Focused on performance to ensure the launcher does not impact your racing experience.
* **Continuous Updates**: Stay informed with the latest racing events and server status through our integrated announcement system.
